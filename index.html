Projects

In Progress

Araxxor Drop Logger

Using the flexibility of Java’s Swing and Abstract Window Toolkit along with MySQL for core tabular relations, I made a program to log drops (loot from successful kills) and show statistics (personal drop rates, total drop counts, drop frequencies) for one of RuneScape 3’s most profitable PvM bosses, Araxxor.

GitHub Repository: https://github.com/swoldemi/Araxxor-Drop-Logger

Completed

GanoGraphic

GanoGraphic is a MATLAB program that generates steganographic images. Along with the steganography, a GPG encrypted decryption key is written to the users local disk so that they may send the image and the key to an authorized recipient.

GitHub Repository: https://github.com/swoldemi/GanoGraphic/

A TensorFlow image classifier

In this project, I retrain Google’s Inception v3 convolutional neural network (10K epochs) to classify the faces of world leaders.

GitHub Repository: https://github.com/swoldemi/World-Leaders-Image-Classification

Ongoing

NervBot - A Discord bot

Inspired by HarmonBot, NervBot is a personal assistant that applies a variety of APIs to service useful tools and my hobbies. My goal in this project is to present information in a user friendly manner, manage data, learn how different programming interfaces service content to developers, and, most imporantly, to have fun!

GitHub Repository: https://github.com/swoldemi/nervbot

Twitter Sentiment Analysis

In this project, I aim to learn more about natural langauge processing using the Python libraries scikit-learn and nltk. To do so, I collect data by streaming live Tweets, based on a keyword, and with the help of these amazing libraries, analyze a generalized sentiment about the keyword.

GitHub Repository: https://github.com/swoldemi/Twitter-Sentiment-Analysis
